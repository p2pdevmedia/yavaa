## Goals

- Mobile first
- Fast onboarding
- Human language
- Trust driven
- Minimal friction
- Shared design language across Web, iOS and Android

## Product Direction

Yavaa is a marketplace for connecting workers with contractors in a fast, trusted and location-aware way.

Visual inspiration:
- Airbnb
- Uber
- Mercado Libre
- Rappi

Primary UX priorities:
- Trust
- Simplicity
- Speed
- Clear flows
- Real identity verification
