# Yavaa Product Principles

## Core Principles

- Mobile first
- Fast onboarding
- Human language
- Trust driven
- Minimal friction
- Location aware
- Real identity verification
- Few clicks
- Simple forms
- Clear actions

## UX Goals

The app should feel:

- Fast
- Safe
- Modern
- Friendly
- Lightweight

The app should NOT feel:

- Corporate
- Administrative
- Enterprise-heavy
- Slow
- Complex

## Visual Direction

Inspired by:

- Airbnb
- Uber
- Mercado Libre
- Rappi
- Linear

## Main Product Goals

- Connect workers and contractors instantly
- Make onboarding extremely fast
- Create trust between both sides
- Use geolocation heavily
- Reduce communication friction
- Allow quick hiring
