# Forms

## Inputs

- Height: 56px
- Radius: 18px
- Background: white
- Border: #E5E5E5

## Labels

- Small uppercase optional
- Secondary text color

## UX Rules

- Forms should feel lightweight
- Avoid dense enterprise forms
- One main action per screen
- Use progressive disclosure

## Validation

Always show:
- error state
- success state
- loading state
