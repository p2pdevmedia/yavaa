# Worker Onboarding Flow

## Goal

Allow a worker to create a verified profile in less than 2 minutes.

---

# Step 1 — Name

Fields:
- Name
- Last name

Validation:
- required
- max 40 chars

---

# Step 2 — DNI

Fields:
- DNI number

Validation:
- numeric only
- valid argentina DNI length

---

# Step 3 — DNI Photos

Required:
- front photo
- back photo

Accepted:
- jpg
- png
- heic

---

# Step 4 — Address

Features:
- map pin
- draggable pin
- GPS autofill
- autocomplete address

---

# Step 5 — Categories

Multi-select categories:
- construction
- cleaning
- classes
- plumbing
- gardening
- painting
- moving

---

# Success

Show:
- verification pending
- nearby jobs
- trust badges
