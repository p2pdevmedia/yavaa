# Client Home UX Priority

## Important UX Decision

The client home screen should have ONE dominant primary action.

Primary CTA:
- Publish a job

Reasoning:
- reduces cognitive load
- faster onboarding
- clearer UX
- feels more like Uber/Airbnb
- avoids decision paralysis

Searching workers still exists, but should be secondary and not visually compete with the main action.

## Visual Direction

The primary card should:
- occupy most of the screen width
- use the main violet color
- contain a single large CTA
- feel immediate and action-oriented
