# Etapa 01 - Base Tecnica y Autenticacion

## Descripcion

Esta etapa prepara la infraestructura minima para que el producto pueda autenticar usuarios y sostener el resto de la aplicacion.

## Objetivo

- Montar la base web.
- Configurar autenticacion.
- Preparar el modelo base de usuarios y roles.
- Definir rutas protegidas.
- Dejar listo el modo activo por rol.

## Incluye

- Next.js.
- TypeScript.
- Supabase Auth.
- Prisma como fuente de verdad del esquema.
- Estructura inicial de permisos.

## Resultado esperado

Un usuario puede registrarse, iniciar sesion y entrar al area correcta segun su rol sin depender de logica manual en el frontend.
