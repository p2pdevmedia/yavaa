# Deliverables

## Required deliverables
- Signup/login/logout + session refresh flows.
- Profile creation/editing and role metadata.
- Contractor onboarding form, DNI uploads, and approval statuses.
- Suspended/blocked user enforcement.

## Completion criteria
- Behavior works end-to-end.
- OpenAPI is updated when APIs change.
- Deterministic automated tests exist for critical flows.
- Permission checks are enforced server-side.
