# Plan de Etapas

Este directorio organiza el plan de puesta en marcha de Yavaa por etapas.

La idea es mantener cada fase separada para que podamos agregar después:

- tareas
- decisiones
- criterios de salida
- riesgos
- tests asociados

Orden recomendado:

1. `etapa-00-alcance-mvp`
2. `etapa-01-base-tecnica-autenticacion`
3. `etapa-02-datos-y-permisos`
4. `etapa-03-flujo-cliente-contratista`
5. `etapa-04-comunicacion-y-archivos`
6. `etapa-05-administracion-y-operacion`
7. `etapa-06-deuda-reputacion-y-validaciones`
8. `etapa-07-calidad-y-piloto`

Cada carpeta contiene la descripcion de la etapa y sirve como punto de partida para documentar el trabajo de esa fase.
