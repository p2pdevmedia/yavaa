# Deliverables

## Required deliverables
- Next.js + TypeScript strict mode configured.
- Supabase + Prisma integration working with migrations.
- OpenAPI generation/publishing baseline.
- Playwright + Vitest running in CI.
- Tailwind + shadcn/ui baseline and shared UI primitives.
- Documented design system plan for tokens, layout primitives, and state patterns.
- Seed system and environment setup docs.

## Completion criteria
- Behavior works end-to-end.
- OpenAPI is updated when APIs change.
- Deterministic automated tests exist for critical flows.
- Permission checks are enforced server-side.
