# Etapa 06 - Deuda, Reputacion y Validaciones

## Descripcion

Esta etapa agrega las reglas de control economico y confianza del sistema.

## Objetivo

- Registrar deuda de comision.
- Validar comprobantes.
- Aplicar bloqueos por estado.
- Guardar ratings y reviews.
- Preparar disputas simples.

## Incluye

- Debt tracking.
- Payment proof review.
- Limites de deuda.
- Bloqueos por deuda.
- Calificaciones.
- Reglas basicas de disputa.

## Resultado esperado

Yavaa puede controlar riesgo, reputacion y estados sensibles con reglas claras y auditables.
