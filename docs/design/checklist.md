# Checklist

## Design planning
- [ ] Design direction documented.
- [ ] Shared UI tokens identified.
- [ ] Core component inventory defined.
- [ ] Page templates mapped to product surfaces.
- [ ] Accessible state patterns documented.
- [ ] Handoff expectations captured.

## Alignment
- [ ] Design plan matches product decisions.
- [ ] Design plan matches user-action map.
- [ ] Design plan stays implementation-free.
