- Client
- Contractor
- Admin

- Search providers
- Book services
- Request emergency help
- Chat with providers
- Leave ratings

## Contractor features

- Configure services
- Configure schedules
- Accept bookings
- Manage emergencies
- Chat with clients

## Admin features

- Manage users
- Moderate providers
- Manage bookings
- Review reports

- Availability
